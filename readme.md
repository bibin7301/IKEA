# IKEA Homepage UI Clone

This project is a frontend UI clone of the IKEA website built using HTML, TailwindCSS, and Font Awesome icons. It provides a responsive navigation bar, search bar, product category section, and a clean modern layout similar to the official IKEA homepage.
